es6新增的数组的方法

filter过滤 ;把符合条件的放进新数组中；原有数组不发生改变；

let  arr = [100,80,56,101];

    let newArr  = arr.filter((item,index)=>{

 return item>70;

    })

console.log(newArr)//[100,80,101]

console.log(arr);//[100,80,56,101]

find查找 :查找；从左到右一次进行查找；找到符合条件的第一项，并且把这一项返回；结束整个函数

let  arr = [34,80,56,101,100];

     let val = arr.find((item,index)=>{

       return item>90;

    });

    console.log(val);//101

some : 返回一个布尔值；如果有符合条件的返回true，如果没有找到符合条件的，返回是false；
如果有符合条件的，那么不再循环，直接停止整个函数；返回true；
如果没找到，会循环所有项，并且返回false

   let  arr = [34,80,56,101];

    let a = arr.some((item,index)=>{

        return item>80;

    });

    console.log(a);//true

every :返回一个布尔值；true或false；必须要求每一项都返回true，整体结果返回true；只要有一个是false，整体结果就是false；

 let  arr = [34,80,56,101];

    let t = arr.every((item,index)=>{

        return item>34;

    })

    console.log(t);//false

includes : 包含；检测是否包含某一项；如果包含返回true，不包含返回false

 let  arr = [34,80,56,101];

 console.log(arr.includes(81));//false

reduce :收敛；求和
reduce第二个参数，代表回调函数中的第一次的prev

 let  arr = [34,80,56,101,999];

     let yy = arr.reduce((prev,next)=>{

         // prev : 输出上一次回调函数的返回值；【第一次除外】

         console.log(prev,next);  

                34 80

                   114 56

                       170 101

                           271 999

         return  prev+next;

     },0);

    console.log(yy);//1270

认识vue

写法不一样：记下来；明白框架的使用
声明式框架：不关心如何实现的
是一个MVVM框架；m:model v:view vm:view-model
model:数据模型
view：视图
view-model：数据和视图之间数据的传递；
MVC：model view controller 数据单项通信，只能通过数据影响视图，
MVVM：model(后端返回的数据)和view是不能直接进行通信的；需要通过view-model来进行数据的传递 model->view：绑定数据view–>model:DOM的监听事件【发布订阅模式】

数据是响应式的，当数据发生变化，会自动影响视图，当视图发生变化，会影响数据–>数据的双向绑定

  • 双向数据绑定 vue利用object.defineproperty这个方法遍历data中的所有属性，给每一个属性增加setter方法和getter方法；当数据发生变化时，会触发set方法，当获取数据时，会触发getter方法；
  • object.definedproperty在IE8以下不兼容的 是es5新增的方法，data中所有的属性最终被vm这个实例所代理；
指令指令指令指令指令指令指令指令指令 含义
v-model 把input的value的值和data的某个属性进行绑定； v-model一般用于表单元素；复选框 单选框 下拉框
v-text 把数据绑定DOM元素中，具有响应式；并且会把之前的进行覆盖
v-html 可以识别标签
v-if 当属性值为true，则显示当前元素，为false，那么显示v-else的元素，组合控制的时DOM元素的存在，不是控制display：none
v-show 控制元素的显示隐藏；通过控制css的display属性来实现显示和隐藏
v-bind 缩写: 用于绑定动态的属性 可省略v-bind 直接前面加一个冒号(:)就可取到动态的属性，一般用于img标签上的src前面
v-cloak 用于元素加载数据比较多时，vue解析时间比较长；为了防止{{}}的出现，给该元素加上v-cloak属性；直到vue中把DOM数据插入到真实DOM中时，让当前的元素显示；
